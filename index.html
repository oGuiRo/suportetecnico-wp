🚀 Conheça o Centro de Comando

Muito mais que um repositório de links. Este é um ecossistema completo de utilitários desenvolvidos sob medida para acelerar o diagnóstico, simplificar configurações complexas e elevar drasticamente a produtividade nos atendimentos do Web Posto.

⚡

FTP Quality Ultra-Rápido

Acesso instantâneo aos arquivos e instaladores S3. A ferramenta utiliza Prefetching e Cache Inteligente: os arquivos são baixados silenciosamente de forma oculta assim que você abre a página inicial, eliminando qualquer tela de carregamento na hora da busca.

🛠️

Validador XML / JSON

Indispensável para análise de integrações e notas. Cole o código minificado e o sistema aplica a Organização automática, organizando as indentações, quebrando linhas corretamente e detectando erros de sintaxe em milissegundos.

🔐

Conversor de Certificados PFX

Descomplica a conversão de certificados .PFX para .CRT (utilizados na criação de API no portal Developers dos bancos ). O script processa nativamente o arquivo .PFX e extrai a chave pública (.CRT) e privada (.KEY) de forma imediata e totalmente segura no navegador (não expondo o certificado do cliente a qualquer risco da Web).

📱

Configurador PAY & Senha

Gera na hora o QR Code formatado para pareamento rápido de terminais. Inclui o motor de Senha Dinâmica original, atualizado em tempo real sincronizado com o relógio.
