Suporte Técnico - Web Posto

A ideia começou apenas para realizar um FrontEnd simples para o FTP de arquivos para download, porém, vendo os utilitários de uso diário do Suporte, resolvi transformar a ideia em um site unificado com coisas utilizadas no dia a dia ou com certa frequência para facilitar e agilizar os atendimentos.
Não é nada perfeito, a ideia é simples e dei inicio conforme as necessidades diárias e com base no que conheço e consigo.

Qualquer dúvida, sugestão ou bug, pode entrar em contato comigo que estou a disposição para ajudar.
