# 🚀 Suporte Técnico - Web Posto

Um projeto criado para centralizar ferramentas e utilitários utilizados no dia a dia do suporte técnico do **Web Posto**, tornando os atendimentos mais rápidos, simples e eficientes.

## 📌 Sobre o projeto

A ideia surgiu inicialmente apenas para disponibilizar um ambiente simples para download de arquivos via FTP. Com o tempo, percebi que diversos procedimentos e ferramentas eram utilizados com frequência durante os atendimentos, então decidi reunir tudo em um único lugar.

O objetivo do projeto é facilitar o trabalho do suporte, disponibilizando recursos que ajudam nas tarefas rotineiras e agilizam a resolução de problemas.

## ⚙️ Funcionalidades

* 📁 Área para download de arquivos;
* 🔎 Ferramentas e utilitários para o dia a dia do suporte;
* 🛠 Recursos voltados para atendimento e troubleshooting;
* 📄 Consultas e informações utilizadas com frequência;
* 🚀 Interface simples e de fácil acesso.
